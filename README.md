# Proyecto DentalCloud Pro

Sistema de gestión odontológica integral desarrollado para la asignatura INF246.

## Características
- **Backend:** NestJS con arquitectura modular.
- **Frontend:** React con Vite y diseño responsivo.
- **Auditoría:** Registro de logs (usuario, IP, evento, timestamp).
- **CRUD:** Pacientes con eliminación lógica.
